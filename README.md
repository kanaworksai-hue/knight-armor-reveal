# Knight Armor Reveal

An interactive double-video reveal demo. Move the pointer over the scene to reveal the training-clothes version beneath the armored knight.

The page is a static HTML/CSS/JavaScript build designed for GitHub Pages.

## Files

- `index.html` - interactive page and animation logic
- `armor.mp4` - base video layer
- `training.mp4` - masked reveal video layer
- `background.mp3` - looped background music
- `poster.jpg` - loading poster image

## Local Preview

Serve the folder with any static server and open `index.html`.
